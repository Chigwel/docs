## Overview

This page describes the current commission model and payment availability for selling skills on SkillDock.

## Commission model

For sold skills, SkillDock commission is **50%** of the sale price.

- Provider share: **50%**
- SkillDock commission: **50%**

### Payments

- TON payments are currently available.
- Stripe Connect will be implemented soon.

For questions about sales, payouts, or pricing setup, contact:

- support@skilldock.io

## Enterprise licence

Enterprise access can be provided for teams that need private team skills shared with team members.

### What we can provide

- Team member access to private team skills
- Team-level/private distribution setup
- Support for access configuration requests

### Request access

To request enterprise/private team access, email:

- support@skilldock.io

Include (recommended):

- Team/namespace name
- Requested member emails
- Which private skills should be shared
- Short use case / organization info
